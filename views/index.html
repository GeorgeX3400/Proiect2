Graph Editor

Editor instructions:

  • Adding a node:

    You must pick a name for your node (the name must have at most 3 characters)

    After choosing the name, click the canvas on the place you want your node to be drawn.

  • Deleting a node:

    Write the name of the node you want to delete; The node will be deleted along with its edges.

  • Adding an edge:

    Write the names of the 2 nodes you want the edge to be between (the nodes must both be in the graph).

  • Deleting an edge:

    Write the names of the 2 nodes of the edge you want to delete.

  • Running BFS:

    Unfortunately, due to the developer's laziness and stupidity, this functionality does not work. :(
